# Review Playbook Skill

## Use This Skill When

- You are reviewing a generated source-derived skill.
- You need to decide whether a source is strong enough to change agent behavior.
- You need to keep one-off context separate from reusable rules.

## Core Guidance

- Prefer the smallest useful artifact.
- Preserve source evidence before promoting guidance.
- Use a full skill only when the source has structure, examples, and repeatable decisions.
- Fold weak but useful material into an existing skill instead of creating a duplicate skill.

## Operating Rules

- Do not turn a single meeting into a full skill by default.
- Treat unsupported claims as pending review items.
- Ask for human review before changing core guidance.
